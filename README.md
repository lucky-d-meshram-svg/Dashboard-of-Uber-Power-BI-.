## 📌 Overview  
This project is an interactive dashboard built using Power BI to analyze Uber ride data. It provides clear insights into bookings, revenue, distance, and ratings.

## 📈 Key KPIs  
- Completed Bookings: 93K  
- Lost Bookings: 57K  
- Revenue: 52M  
- Total Distance: 2.51M  
- Average Distance: 24.64  
- Customer Rating: 4.40  
- Driver Rating: 4.23  

## 📊 Features  
- Monthly booking and revenue trends  
- Vehicle-wise performance analysis  
- Top pickup and drop locations  
- Interactive filters for better data exploration  

## 🛠️ Tools Used  
- Power BI  
- Data Visualization  
- DAX (for calculations)  
---

## 🚀 Purpose  
The goal of this project is to transform raw ride data into meaningful insights to understand patterns and support better decision-making.
